# QuickBill Desk – Professional Invoicing & Client Management

**Full-stack invoicing platform** built with React, Node.js, MongoDB.   
Source code included in this repo.

## Features
- Invoice creation, PDF generation, email delivery
- Client management with search & filters
- Bulk operations, dark mode, backup/restore
- RESTful API with JWT auth

## Tech Stack
- Frontend: React, TypeScript, Vite, Tailwind, Shadcn/ui
- Backend: Node.js, Express, MongoDB, Mongoose
- Other: PDFKit, Nodemailer, Docker
